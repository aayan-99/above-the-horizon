# Above The Horizon

This is a simple react router bassed project to explore the primaty concepts of React like state, props, react-router, etc.

## [Demo Link](https://above-the-horizon.vercel.app/) 🔗


## Technologies used 🛠️

- **HTML** 🚀
- **CSS** 🚀
- **JavaScript** 🚀
- **Tailwind CSS** 🚀
- **React JS** 🚀



## How To Use 🔧

From your command line, first clone this repo:

```bash

# Clone this repository
$ git clone https://github.com/aayan-99/above-the-horizon.vercel.app

# Go into the repository
$ cd above-the-horizon

# Remove current origin repository
$ git remote remove origin

```

Then you can install the dependencies

Using NPM:

```bash

# Install dependencies
$ npm install

```
